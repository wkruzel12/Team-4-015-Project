# Bistro Burnett Online Ordering System

A beginner-friendly Flask + SQLite class project prototype.

## Tech Stack
- Python
- Flask
- HTML/CSS
- SQLite

## Run Locally
1. Create and activate a virtual environment (optional but recommended).
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the app:
   ```bash
   python app.py
   ```
4. Open your browser to:
   ```
   http://127.0.0.1:5000
   ```

## Main Workflow
1. Enter phone number
2. Enter customer name
3. Choose category (Sandwich or Wrap)
4. Build order
5. Review order
6. Verify/payment notice
7. Confirmation
8. Orders management (search/filter/update status)

## Database Operations Included
- Insert a new order
- Retrieve all orders
- Filter orders by phone number
- Update order status
